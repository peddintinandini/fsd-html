Full Stack

Full Stack developers must have knowledege of an entire, technology stack,i.e.,the set of techologies that are used to bulid an end-to-end application quickly and efficienctly.For example, if they want to bulid an application using themean stackthey should know how to work with MongoDB,Express,Angular and Node

Full Stack development refers to the end to end application software development ,including the front end back end.The front end consists of the user interface,and the back end takes care of the business logic and application workflow.
